come funziona
📂
1. Carica i file STL
File A = scansione di riferimento (master). File B = scansione da verificare. Entrambi contengono più cilindri (impianti).
🔄
2. Allineamento ICP
Algoritmo Kabsch + SVD allinea globalmente B su A. L'allineamento è invariante a traslazione e rotazione.
📐
3. Misurazioni
Per ogni cilindro: deviazione XY (piano occlusale), Z (altezza), rotazione assiale. Tutto in micrometri.
🏆
4. Voto 0–100
Score ponderato: XY peso 45%, Z peso 35%, asse rotazione 20%. Penalità esponenziale oltre le soglie cliniche.

Con consenso classifica
Compili nome, città, paese, tipo struttura e accetti di condividere il risultato → vedi il voto e la tua misurazione entra in classifica pubblica (solo città, paese, tipo e voto).
formula voto
Score = 100 − penalità_totale
penalità_XY = Σ(|D|_xy_i / soglia_ottimo) × 45 / n_coppie
penalità_Z = Σ(|dZ_i| / soglia_ottimo) × 35 / n_coppie
penalità_Ax = Σ(|asse_i| / soglia_ax_ottimo) × 20 / n_coppie
(ogni termine clamped a 0–45/35/20; score finale clamped 0–100)
